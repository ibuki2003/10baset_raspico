# Raspberry Pi Pico 10BASE-T stack

The goal of this project is to communicate with generic ethernet equipments.

## Features / TODO

- [x] 10BASE-T TX
- [x] 10BASE-T RX
- protocols
  - [ ] ARP
  - [ ] IP
  - [ ] DHCP
  - [ ] ICMP
  - [ ] TCP
  - [ ] HTTP
  - [ ] UDP
- [ ] blinking status lamps on the connector
- [ ] multiple NIC, behave as switching hub
